# Guess the Number (Угадай число)

Консольная игра на Go, в которой игроку нужно угадать случайное число за ограниченное количество попыток.

## Возможности
- 3 уровня сложности (Easy / Medium / Hard)
- Подсказки: холодно / тепло / горячо
- Цветной вывод в терминале (ANSI escape-коды)
- Проверка и валидация пользовательского ввода
- Сохранение результатов игр в файл `results.json`
- Возможность сыграть несколько раз подряд
